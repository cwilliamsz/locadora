Hands on Work - Locadora de Veículos

1) Abrir o programa de um pacote de servidores (indica-se XAMPP)
2) Iniciar no pacote de servidores o Gerenciador de Bando de dados MySQL, clicando em start (o banco de dados se chama locadora_hand_on_work).
3) Selecionar no GitHub o arquivo do sistema desenvolvido.
4) Abrir o arquivo em um ambiente de desenvolvimento integrado ou IDE (indica-se Visual Studio Code).
5) Acessar a aba Terminal e clciar em New Terminal. Alternativamente Ctrl+Shift+'.
6) Digitar "npm start" e aperter enter.
7) Acessar o navegador web de preferência (indica-se Google Chrome ou Opera).
8) Digitar na barre de endereços: "localhost:3000".
9) Digitar o login handsonwork
10) Digitar a senha handsonwork1
